# GitItUp — Changelog

## [0.4.5] — 2026-03-25

### Features & Improvements
- **Official Branding**: Integrated new **`GitItUpLogo.ico`** and **`GitItUpLogo.png`** as official icons and UI brand. 🎨
- **Custom CSS Modding**: Added a new settings section that allows users to import and persist external `.css` files for advanced personalization.
- **Smart Scoring System**: Refined XP progression where Commits (1XP), Pushes (25XP), and Merges/PRs (50XP) provide different rewards.
- **Robust Multi-Hook Automation**: Now automatically configures global hooks for `post-commit`, `pre-push`, and `post-merge`.
- **Polish & UI**:
    - Added a "Screen Flash" effect when receiving Git events to confirm detection.
    - Added an "XP Pop" animation for more visual "juice".
    - Added **`Instalar_Hooks.bat`** as a utility to fix Git configurations manually.
